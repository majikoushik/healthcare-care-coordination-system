# ADR-0002: React Frontend Choice

## Status

Accepted

## Context

The UI must be a professional healthcare operations portal with strong TypeScript support.

## Decision

Use React, TypeScript, Vite, React Router, TanStack Query, React Hook Form, Zod readiness, and Axios.

## Consequences

The frontend is fast to run locally and aligned with modern enterprise dashboard expectations.

## Alternatives Considered

Angular was not selected because AGENTS.md explicitly prefers React for this portfolio project.
