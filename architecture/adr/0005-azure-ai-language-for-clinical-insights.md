# ADR-0005: Azure AI Language For Clinical Insights

## Status

Accepted

## Context

Clinical note insight readiness should use a healthcare-specific Azure AI direction.

## Decision

Prepare for Azure AI Language Text Analytics for Health through a provider abstraction.

## Consequences

The design can integrate Azure AI later without requiring credentials locally.

## Alternatives Considered

Azure OpenAI was deferred because the first healthcare AI capability should align to Azure AI Language for Health.
