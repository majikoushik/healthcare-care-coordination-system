Implement Epic 0: Repository Foundation based on AGENTS.md.

Create the initial enterprise-grade foundation for the `healthcare-care-coordination-system` repository.

Follow the architecture, quality standards, healthcare privacy guidance, folder structure, Azure-readiness direction, documentation expectations, testing expectations, React frontend standards, SQL Server + Cosmos DB persistence direction, Azure AI Language readiness, and agent workflow rules defined in `AGENTS.md`.

Epic 0 Goal:

Create a clean, professional, Solution Architect-grade repository foundation for a healthcare care coordination platform.

This project must not look like a simple CRUD app or basic appointment booking system. It should look like a cloud-native healthcare care coordination platform designed by a senior Solution Architect.

Project stack:

* Backend: .NET 8 / ASP.NET Core Web API
* Frontend: React + TypeScript + Vite
* Relational database: SQL Server / Azure SQL
* NoSQL database: Azure Cosmos DB for flexible healthcare documents and events
* Azure AI readiness: Azure AI Language / Text Analytics for Health
* Cloud target: Azure Container Apps, Azure Static Web Apps, Azure SQL, Azure Cosmos DB, Azure Key Vault, Azure Application Insights, Azure Log Analytics, Azure Service Bus
