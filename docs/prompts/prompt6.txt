Mock AI provider behavior:

The mock provider should use deterministic keyword-based extraction from synthetic note text.

Example:

If note contains:

* "blood sugar" or "glucose" → entity category `Test` or `Condition`
* "fatigue" → entity category `Symptom`
* "medication adherence" or "missed medication" → risk indicator `MissedMedication`
* "follow-up" → entity category `FollowUp`
* "diet" or "exercise" → entity category `Lifestyle`

Keep the mock logic simple, deterministic, and testable.

Do not use machine learning libraries.

Do not call external APIs.

Do not require Azure credentials.
