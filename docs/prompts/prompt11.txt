Epic 10 Goal:

Add security, privacy, and RBAC readiness across the healthcare care coordination platform without implementing a full production identity system.

This epic should demonstrate how the system would support secure healthcare workflows, role-aware access, safe data handling, privacy-by-design, and future Azure Entra ID integration.

This project must remain a portfolio demo using synthetic data only.
