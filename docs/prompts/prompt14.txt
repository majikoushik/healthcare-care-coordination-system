Epic 13 Goal:

Create a professional Azure deployment blueprint for the healthcare care coordination platform.

This epic should not require real Azure credentials to run locally. It should provide infrastructure-as-code templates, deployment documentation, environment strategy, security guidance, and cloud architecture diagrams showing how this system would be deployed on Azure.

Target Azure architecture:

* Azure Static Web Apps for React frontend
* Azure Container Apps for .NET backend APIs and worker services
* Azure Container Registry for container images
* Azure SQL Database for Patient, Provider, and Appointment data
* Azure Cosmos DB for Care Plans, Clinical Insights, Follow-up Tasks, Notifications, and Audit Events
* Azure AI Language for future Text Analytics for Health integration
* Azure Key Vault for secrets
* Azure Application Insights for telemetry
* Azure Log Analytics Workspace for centralized logs
* Azure Service Bus for future event-driven communication
* Azure API Management as optional future gateway
* Azure Front Door as optional future edge layer
